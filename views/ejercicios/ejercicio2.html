Ejercicio 2 — Matrices y sistemas

Sea la matriz A = [[2,3],[1,4]] y el vector b = (7,9).

  1. Calcula el determinante de A.
  2. Resuelve el sistema A x = b (encuentra x = (x1,x2)).
  3. Determina si el vector v = (5,8) pertenece al span de las columnas de A (es decir, ¿existen α,β tal que α·col1 + β·col2 = v?).
