Lección 1: Introducción a los Vectores

Un vector es una cantidad que tiene magnitud y dirección. Se representa comúnmente como una flecha en un plano o espacio.

Representación

Un vector en 2D se escribe como: 

v = (x, y)

Magnitud de un vector

La magnitud se calcula así:

|v| = √(x² + y²)

Ejemplo

Para el vector v = (3, 4):

|v| = √(3² + 4²) = 5
